## Overview

First official session logged as a Maker Mentor at iLab. The day was focused on getting familiar with the space, meeting the current batch of makers, and understanding the projects in progress. Good energy in the room with a mix of beginners and intermediate-level makers working across hardware and software.

<!-- more -->

## Mentees / Participants

- Ali – building an automatic plant watering system using Arduino
- Fathima – working on a mobile app for campus event tracking
- Rahul – exploring 3D printing for a custom drone frame

## Projects Guided

- **Auto Plant Watering System** — helped Ali wire up a soil moisture sensor and write the threshold logic in Arduino IDE
- **Campus Event App** — discussed UI flow and helped Fathima set up a Firebase Realtime Database connection
- **Drone Frame Design** — reviewed Rahul's Fusion 360 sketch and suggested wall thickness adjustments for print strength

## Activities

- Reviewed Arduino wiring diagrams with Ali and fixed a short circuit on the breadboard
- Walked Fathima through Firebase authentication setup
- Discussed FDM print settings (infill %, layer height) with Rahul for structural parts

## Highlights

- Ali's moisture sensor gave its first correct reading after fixing the wiring — great moment
- Rahul's frame design is solid, just needs minor tolerancing tweaks before printing

## Challenges

- Fathima's app has a state management issue — needs more time next session
- Need to source a better moisture sensor module, current one has inconsistent readings
